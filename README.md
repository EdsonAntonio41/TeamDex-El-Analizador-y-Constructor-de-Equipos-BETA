# TeamDex-El-Analizador-y-Constructor-de-Equipos-BETA
Esta web está hecha por un estudiante amante de Pokémon, pero bastante pésimo armando equipos. Su objetivo es ayudar a las personas que quieran jugar Pokémon Champions sin complicarse tanto, ofreciendo un espacio donde apoyarse para formar o copiar equipos ya hechos, e ir aprendiendo a su propio ritmo.
